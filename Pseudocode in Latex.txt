# Pseudocode in Latex

## Description
This project aims to create a tool that takes the input of a C++ function and returns a pseudocode snippet written in LaTeX that can be used inside a LaTeX document.
## Specifications
A web application with a text area where the user can enter the function, hit a submit button, and get the output that can be copied for further use.
Include as much language support as possible and detailed documentation of all the accepted keywords and how they will get converted. Also, remember to convert the comments to LaTeX.
